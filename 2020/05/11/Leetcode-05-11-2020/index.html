3.2 Solution#

Normally, this question belongs to the sliding window problem. We use two pointer to maintain a sub array which with no repeating characters.

Whenever we find the character which has been occured before, we maintain a map to record the last occurence of that character. Whenever one repeat character found, we need to update our left pointer to maintain the non-repeating character array $array[left,i]$

5.2.1 Dynamic Programming#

The recurrence formula is $dp[i][j] = 1 \quad if \quad s.charAt(i) == s.charAt(j)\quad \And\And \quad dp[i+1][j-1] == 1$

5.2.2 Expand Around Center#

A faster way is to expand the palindrome list as long as possible with center in each span. For example

Abba. There are $2\cdot n-1$center. There are A, b, b, a, the center between Ab, bb, ba.
